# Create custom server using HTTP

A lightweight and customizable HTTP server built using **Node.js**, **Express.js**, and managed with **Nodemon** for a seamless development experience.

---

## 🚀 Features

* **Express.js Routing:** Clean and robust routing system.
* **Custom HTTP Methods:** Handles custom requests and responses efficiently.
* **Live Reloading:** Integrated with `nodemon` to automatically restart the server on file changes.
* **Environment Configuration:** Easy to configure ports and settings.

---

## 🛠️ Prerequisites

Before running this project, ensure you have the following installed on your local machine:

* [Node.js](https://nodejs.org/) (v14.x or higher recommended)
* [npm](https://www.npmjs.com/) (comes pre-installed with Node)
